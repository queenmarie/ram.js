1. ram.('selector')

The ram object. Use `ram('selector')` for quick DOM access when built with the DOM module.
@returns {Object} The ram object, run through `init`
Example: ram.('li')


2. object.isArray()

Determine if an object is an `Array`.
@param {Object} object An object that may or may not be an array @returns {Boolean} True if the parameter is an array.
Example: ram.isArray()


3. object.toArray()

Convert an `Array`-like collection into an `Array`.
@param {Object} collection A collection of items that responds to length @returns {Array} An `Array` of items.
Example: ram.toArray()


4. object.isNumber()

Determines if an object is a `Number`.
@param {Object} object A value to test @returns {Boolean} True if the object is a Number.
Example: ram.isNumber()


5. object.bind()

Binds a function to an object.
@param {Function} fn A function @param {Object} object An object to bind to @returns {Function} A rebound method
Example: ram.bind()


6. object.addDetection()

Used to add feature-detection methods.
@param {String} name The name of the test @param {Function} fn The function that performs the test.
Example: ram.bind()


7. object.detect()

Run a feature detection name.
@param {String} name The name of the test @returns {Boolean} The outcome of the test
Example: ram.detect()


8. The ram Class


							User = ram.Class({
							initialize: function(name, age) {
							this.name = name;
							this.age  = age;
						}
					});

Example: new User('Ritwick', '52');


6. Mixins

Objects can be embedded within each other:

 
				
				MixinUser = ram.Class({
				include: User,
				
				initialize: function(log) {
				this.log = log;
			}
		});


7. Inheritance

Pass an object to `ram.Class` to inherit from it.


			SuperUser = ram.Class(User, {
			initialize: function() {
			this.$super('initialize', arguments);
		},
		
		toString: function() {
		return "SuperUser: " + this.$super('toString');
	}
});


8. Enumrable module

This is bound to DOM objects: 


		global('p').each(function() {
		// `this` contains a DOM element
	});

Iterates using a function over a set of items Example: 

ram.enumerable.each([1, 2, 3], function(n) {
		console.log(n);
	});

@param {Object} enumerable A set of items that responds to `length`
@param {Function} callback The function to run
@param {Object} [context] An optional parameter to determine `this` in the callback
@returns {Object} The passed in enumerable object
